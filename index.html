Import dataset

First let’s import a dataset. We will use open-access data from the CIBMTR: HLA haploidentical versus matched unrelated donor transplants with post-transplant cyclophosphamide based prophylaxis. https://www.cibmtr.org/ReferenceCenter/PubList/PubDsDownload/Pages/default.aspx

library(tidyverse)
library(gtsummary)
library(haven)
df <- read_sas("gs1801_v2.sas7bdat",NULL)

The skim() function is another great tool to explore your dataset. Descriptive statistics are given for each variables, including number of missing values. Now that we know the dataset is looking nice and clean (particularly variable names!), let’s build the table using the great {gtsummary} package.

Build Table 1 (Patient and transplant characteristics)

df %>% 
  transmute( # transmute() allows us to both select and transform variables
    Sex=ifelse(sex==1,"Male","Female"), #ifelse() is convenient to recode binary variable
    Age=age,
    `Performance status (Karnosfky Scale)`=ifelse(kps==1,"90-100","<90"), #Back ticks allow you include spaces in variable names
    `Disease type`=recode(disease,"1"="AML","2"="ALL","3"="MDS")
  ) %>% 
    tbl_summary(
    statistic = list(
      all_continuous() ~ "{median} ({p25}, {p75}) \n {min}-{max}",
      all_categorical() ~ "{n} ({p}%)"
    )
  ) %>% 
  bold_labels()

Split and compare groups in Table 1

df %>%
  transmute(
    Sex = ifelse(sex == 1, "Male", "Female"),
    Age = age,
    `Performance status (Karnosfky Scale)` = ifelse(kps == 1, "90-100", "<90"),
    `Disease type` = recode(
      disease,
      "1" = "AML",
      "2" = "ALL",
      "3" = "MDS"
    ),
    `Donor group` = ifelse(main == 1, "Haplo PTCy/CNI/MMF", "MUD PT-Cy/CNI/MMF")
  ) %>%
  tbl_summary(
    by = `Donor group`,
    statistic = list(
      all_continuous() ~ "{median} ({p25}, {p75}) \n {min}-{max}",
      all_categorical() ~ "{n} ({p}%)"
    )
  ) %>%
  bold_labels() %>%
  add_p() %>%
  add_stat_label()
